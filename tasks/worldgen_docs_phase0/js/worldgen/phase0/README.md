# `js/worldgen/phase0/`

This is a runtime skeleton for Phase 0.

## Modules
- `index.js` — public export surface
- `master-seed-generator.js` — top-level orchestration
- `deterministic-rng.js` — stable RNG helpers
- `world-profile-synthesizer.js` — latent axis generation and shaping
- `world-tone-synthesizer.js` — readable world tone label generation
- `derived-tendencies.js` — tendency package synthesis
- `subseed-deriver.js` — deterministic phase namespace seeds
- `validation.js` — Phase 0 validation rules
- `contracts.js` — runtime contract checks
- `debug/` — summaries and snapshots
- `adapters/` — future bridge adapters to orchestration
